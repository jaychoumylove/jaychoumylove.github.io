统计类的查询

大多是统计类的函数的运用，主要有：

  • COUNT(): 计数
  • SUM(): 求和
  • AVG()：求平均值
  • MIN()：取最小值
  • MAX()：取最大值

count 里面一般传一个字段或者是一个数值，个人喜欢传 1
